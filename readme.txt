A plugin to link your rmoov webmaster account to your wordpress blog for easy link cleanup

== Description ==

A rmoov plugin which allows you to use the rmoov API help process link removal requests. The plugin will list all link removal requests for your blog from rmoov users and with one click it will deal with them all. This will speed up link removal processing and help to relieve the webmaster from the worry of being flooded with email requests.

== Installation ==

Standard installtion required.

1. Upload 'rmoov_Link_Removal.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the plugin in the 'Tools' menu
